# Release Notes

## Version

3.5.4

---

## NEW

* `camera_server` 启动时自动读取光机启动状态
* `camera_server` 启动时输出当前网卡配置信息
* MCU 增加固定版本号机制（格式：V年月日，本次为 **V260325**）

---

## FIX

* 修复相机与 MCU 通讯 ACK 异常问题
* 解决相机采集过程中出现 `-13 / -15` 异常
* 优化异常恢复机制：

  * 当出现 `-13 / -15` 后重启光机
  * 自动恢复上一次的参数配置
* 修复相机重新上电后，在 `camera_server` 不重启情况下无法正常采集的问题
* 修复标定文件不一致问题：

  * 自动覆盖本地标定文件以保持一致性
